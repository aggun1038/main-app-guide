# main-app-guide

## 项目简介

`main-app-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，旨在以结构化方式整理和展示一系列静态 HTML 资源，方便查阅、分发和维护。

## 目录说明

```
main-app-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（样式、脚本、图片等）
├── docs/           # 附加文档或说明文件
└── README.md       # 本文件
```

- **pages/**：每个独立 HTML 页面以单独文件或子目录形式存放，命名清晰，便于识别用途。
- **assets/**：共用的样式表、JavaScript 脚本、字体、图片等资源，按类型或页面分组。
- **docs/**：额外的使用说明、变更记录或参考文档。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立功能单元，可直接在浏览器中打开运行。页面可能包含演示、工具、信息展示等内容，但均不指向任何具体在线服务或商业产品。

归档原则包括：

- 每个页面应尽量自包含，减少外部依赖。
- 如需引用外部资源，请在页面内注明来源。
- 页面内容保持简洁，避免冗余或无关信息。

## 维护说明

- 新增页面时，请将其放入 `pages/` 目录，并在文件名或子目录名中体现页面主题。
- 如有共用资源，请优先放入 `assets/` 下对应子目录，避免重复。
- 修改或删除页面前，请确认不影响其他页面的引用关系。
- 建议在 `docs/` 中记录重要的变更历史或版本说明。
